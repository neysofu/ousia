Usage: ousia <command> [options] [<arguments>]

The available commands are:
* beautify
  Beautify source files.
* build
  Compile a project.
* help
  Show usage information about a command.
* install
  Fetch and install a library.
* run
  Compile and run a script on the fly.
Type "ousia help <command>" for more information about a command.
