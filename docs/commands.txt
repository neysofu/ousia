By writing plugins, you can also add your own commands to Zenzi.

* bug
* help
* install https://github.com/abc/xyz
* repl
* list
* run
* plugins
* profile
* beautify
* update
* version

    curl -sL https://zenzi.org | bash
    zenzi install

zenzi bug


zenzi profile --deterministic .

All this commands are also available in the REPL as zenzi functions.
